API documentation

BrewInfo

Display pop-up with Homebrew Formula info, or open their URL
Caffeine

Prevent the screen from going to sleep
ColorPicker

Show a color sample/picker
Emojis

Let users choose emojis by name/keyword
FadeLogo

Show a fading-and-zooming image in the center of the screen
MenubarFlag

Color the menubar according to the current keyboard layout
MouseCircle

Draws a circle around the mouse pointer when a hotkey is pressed
RoundedCorners

Give your screens rounded corners
Seal

Pluggable launch bar
Seal.plugins.useractions

Allow accessing user-defined bookmarks and arbitrary actions from Seal.
SendToOmniFocus

Handles "send current item to OmniFocus" for multiple applications
TextClipboardHistory

Keep a history of the clipboard, only for text entries
ToggleScreenRotation

Toggle rotation on external screens
ToggleSkypeMute

Provide keybindings for muting/unmuting Skype or Skype for Business
UniversalArchive

Handle "archive current item" for multiple applications using the same hotkey
URLDispatcher

Route URLs to different applications with pattern matching
WiFiTransitions

Allow arbitrary actions when transitioning between SSIDs
