docs » HoldToQuit

Instead of pressing ⌘Q, hold ⌘Q to close applications.

API Overview

  • Variables - Configurable values
    • defaultHotkey
    • duration
    • hotkeyQbj
    • timer
  • Methods - API calls which can only be made on an object returned by a constructor
    • bindHotkeys
    • init
    • killCurrentApp
    • onKeyDown
    • onKeyUp
    • start
    • stop
